# ParseTorrent

ParseTorrent parses a .torrent file and returns a map

## Installation

Add parse_torrent to your list of dependencies in `mix.exs`:

    def deps do
      [{:parse_torrent, "0.1.0"}]
    end

## Usage
    ParseTorrent.parse("my_path/my_file.torrent")
